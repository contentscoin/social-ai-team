# 카드뉴스 팩 (내장 v1)
한국형 카드뉴스(인스타 캐러셀·네이버 카드형) 다중 카드 설계 문법. claude-svg 레인 전용.
원칙: **카드 한 장 = 메시지 한 개.** 시리즈 전체가 한 사람이 디자인한 것처럼 보여야 한다.

## 카드 구성 공식 (N장 기준)
1. **표지 (1장)** — 후킹 장치 하나만: 질문형("아직도 ~하세요?") / 숫자 티저("모르면 손해보는 3가지") / 반전 선언.
   제목 최대 12자 × 2줄. 하단에 "밀어서 보기 →" 스와이프 유도. 브랜드 마크 작게.
2. **본문 (N-2장)** — 카드당 소제목 1개(최대 14자) + 본문 2-3줄(줄당 최대 16자) + 선택적 아이콘/일러스트 1개.
   좌상단에 진행 표시: `01 / 05` 형식 또는 진행 도트. 카드마다 같은 위치.
3. **엔딩 (1장)** — 핵심 요약 1줄 + CTA 1개(저장/팔로우/댓글 유도, 링크 언급) + 브랜드명·핸들.

## 시리즈 일관성 규칙 (전 카드 공통 — 어기면 실패)
- 같은 레이아웃 그리드: 제목 위치·본문 위치·진행표시 위치·여백을 전 카드 동일하게
- 같은 팔레트: 배경 1색 고정 + 액센트 1색. 카드별 변주는 액센트 농도/도형 배치만
- 같은 타이포 스케일: 소제목 56-64px / 본문 34-40px / 진행표시 22px (1080 기준)
- 같은 장식 문법: 표지에 쓴 도형 스타일(원/사선/밑줄)을 본문·엔딩에도 재사용

## 카드당 텍스트 예산 (초과 시 카드 분할)
- 표지 제목 ≤ 24자, 본문 카드 ≤ 80자, 엔딩 ≤ 60자
- 문장은 명사형 종결 또는 짧은 해요체 — 카드에서 만연체는 죽는다
- 숫자·키워드는 액센트 색 + 굵기로 강조 (밑줄 도형 허용)

## 표지 후킹 패턴 뱅크
- 손실 회피: "모르면 손해보는 ___"  - 반전: "사실 ___는 틀렸습니다"
- 궁금증 격차: "99%가 놓치는 ___"  - 공감 호출: "___하는 사람 특징"
- 숫자 약속: "___하는 법 5가지"    - 계절 긴급: "지금 아니면 늦는 ___"

## 스와이프 유지 장치 (본문 카드)
- 카드 끝에 다음 카드 예고 반 문장 ("그런데 진짜 문제는…")
- 진행 표시로 남은 분량 인지시키기
- 마지막 본문 카드에 "저장해두면 좋은 이유" 한 줄

## 채널별 사이즈
- 인스타 캐러셀: 1080×1350(4:5) 권장 — 피드 점유 최대
- 정방형 1080×1080: 네이버·페북 겸용
- 표지만 사진 레인(photo)으로 뽑고 본문은 SVG 카드로 섞는 하이브리드 허용 —
  이때 표지 사진에는 텍스트 오버레이 공간(negative space) 확보 필수

## 출력 계약 (멀티카드 — 렌더러가 파싱)
- 카드 수 N을 지시받으면 정확히 N개의 <svg> 블록을 순서대로 출력
- 각 SVG 사이에 구분 주석 <!--CARD--> 한 줄
- 각 SVG는 동일 캔버스 크기, self-contained, 설명 텍스트 금지

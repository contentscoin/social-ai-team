# Series Variation Matrix (시리즈 변주 매트릭스)

`/reels-script` Phase 7(시리즈 변주)의 풀 버전 레퍼런스. 시리즈 콘텐츠(같은 코너의 다회차, 시즌 캠페인, 멀티 페르소나 광고)에서 **편마다의 차이를 의도적으로 설계**하기 위한 도구입니다. 사용자의 video-production-skills 리포 `series-variation` 스킬에서 소셜 숏폼용으로 발췌·각색했습니다.

## 핵심 개념: LOCKED vs VARIABLE

**LOCKED (모든 회차 공통 — 흔들면 시리즈가 깨짐):**

```yaml
locked:
  brand_assets:        # 로고, 워드마크, CTA 문구, 엔드카드
  format:              # 길이(예: 30s), 비율(9:16), 엔드카드 시간
  look_spec:           # 컬러 팔레트(brand-style.md), 그레이드, 무브먼트 규칙
  narrative_arc:       # 비트 구조와 타이밍 (예: Hook 2s → Context 4s → Core 4s → CTA 2s)
  guardrails:          # 금지 표현(kr-voice-profile.md), 톤 하한/상한
  subtitle_style:      # 자막 위치·서체 규칙 (reels-script 자막 트랙 규약)
```

**VARIABLE (편별 변주 — 7개 차원):**

| # | 차원 | 질문 | 값 예시 |
|---|---|---|---|
| 1 | Target Persona | 누구에게 말하는가 | 신규 잠재고객 / 기존 고객 / 의사결정자 / 실무자 |
| 2 | Location | 어디서 일어나는가 | 실내↔실외, 일상↔비일상, 시간대(새벽/낮/골든아워/밤) |
| 3 | Protagonist | 누가 주인공인가 | 단독 / 2인 / 그룹 / 익명(손·실루엣) / 오브제 중심 |
| 4 | Central Motif | 기억에 남는 시각 어휘 1~2개 | 도구 / 행위 / 풍경 / 신체 디테일 |
| 5 | Emotional Tone | 핵심 정서 (톤 하한~상한 사이) | 신중 / 정성 / 편의 / 품격 / 합리 / 신뢰 |
| 6 | Information Density | 추상 ↔ 직설 (1~5) | 1 분위기만 / 2 시적 자막 / 3 메시지 자막 / 4 데이터·기능 / 5 가격·혜택 |
| 7 | Narrative Perspective | 누구의 눈으로 보는가 | 관찰자 / OTS·시점 / 1인칭 손 / 상징(인물 없음) |

## 매트릭스 관리

시리즈 전체를 한 표로 관리합니다 (예시):

| 편 | Persona | Location | Protagonist | Motif | Tone | Density | Perspective |
|----|---------|----------|-------------|-------|------|---------|-------------|
| V1 | 검토자 | 실내 | 1인 | 펜·노트 | 신중 | 2 | 관찰자 |
| V2 | 담당자 | 다이닝 | 2인 | 악수 | 정성 | 3 | 관찰자 |
| V3 | 기프팅 | 책상 | 익명 손 | 박스 | 품격 | 1 | 1인칭 손 |
| V4 | 실무자 | 오피스 | 1인 | 그래프 | 합리 | 4 | 관찰자 |

## Health Check (매트릭스 완성 후 점검)

- **다양성**: 각 차원에 최소 2개 이상의 값이 등장하는가? (LOCKED 항목 제외)
- **차별성**: 두 편이 6개 이상 차원에서 같으면 사실상 같은 광고. 인접 편(V1↔V2)은 서로 다른 차원이 3개 이상 되게.
- **균형**: Density에 1과 4가 함께 있어야 시리즈 폭이 넓음. 인물 등장/익명, 실내/실외 비율이 한쪽으로 쏠리지 않게.
- **누락 점검**: 빠진 후보를 제안 (예: "야간 컷 없음", "여성 주인공 부족").

## 변주 제안 휴리스틱

- **첫 편은 시리즈의 정중앙**: Density 2~3, 관찰자 관점, 메인 페르소나. 양극단(추상 1, 직설 5)은 2~3편 이후.
- **가장 다른 편 만들기**: 이전 편들의 차원별 평균에서 가장 먼 값을 선택 (전부 "관찰자"였다면 다음은 "1인칭 손" 또는 "상징").
- **대조 쌍 배치**: 추상 편과 구체 편을 인접 배치하면 시청자가 시리즈의 폭을 빨리 인지.
- **타깃→차원 매핑**: 데이터형 의사결정자 → Density 4 / 정서 구매자 → Density 1~2 / B2B 임원 → 단독·2인 + OTS 관점.
- **상징·추상 편은 시리즈당 최대 1~2편**: 많으면 메시지가 흐려짐.
- **Density 5(가격·혜택 직설)는 캠페인 후반·전환 광고 전용.**

## 이 리포에서의 사용법

- `/reels-script` Phase 7: 새 회차 작성 전, 직전 회차 파일(`outputs/videos/`)에서 7개 차원 값을 추출해 이 매트릭스에 행으로 적고, Health Check를 통과하는 변주를 골라 씁니다. 매트릭스는 대본 파일의 `Series note` 필드에 요약 기록합니다.
- `/ad-storyboard`: 시리즈 광고 캠페인이면 스토리보드 간 변주에도 같은 7차원을 적용합니다.
- `/content-calendar`: 시리즈 코너를 캘린더에 배치할 때 편별 Persona·Density 분포를 미리 스케치하면 슬롯 설계가 쉬워집니다.
- `/content-director`: Notes의 "두 달 연속 같은 포맷/톤이면 변주 제안" 체크가 이 매트릭스의 축약형입니다 — 시리즈물이 늘어나면 이 풀 매트릭스로 승격하세요.

# 클로드 디자인(SVG) 팩 (내장 v1)
claude-svg 레인용 디자인 브리프 규칙. 한글 타이포그래피 카드가 이 레인의 주특기다.

## 레이아웃 아키타입 (포맷·목적으로 선택)
1. 히어로 헤드라인 — 훅 1줄(초대형) + 서브 1줄 + 하단 브랜드 마크. 발표/공지/시즌 오픈
2. 스탯 카드 — 큰 숫자(120-180px) + 단위 + 맥락 1줄 + 미니 바/도넛 차트. 성과/데이터 훅
3. 인용 카드 — 따옴표 장식 + 인용문(2-3줄) + 출처. 후기/한줄 철학
4. 체크리스트 — 제목 + 3-5개 체크 항목(아이콘+텍스트). 팁/가이드형
5. 비교 카드 — 좌우 2분할(vs) + 각 측 라벨+포인트. before/after, A vs B
6. 캐러셀 표지 — 좌측 60% 타이포 블록 + 우측 추상 도형, "→ 밀어보세요" 힌트

## 한글 타이포 규칙
- 줄바꿈: 의미 단위로 2-7어절, 조사에서 끊지 않는다 ("라떼 아트를 / 집에서" ○, "라떼 아트 / 를 집에서" ×)
- 크기 위계: 헤드라인 72-110px / 서브 32-40px / 캡션 22-26px (1080 기준), 위계는 3단계까지만
- letter-spacing: 헤드라인 -0.02em, 본문 0. line-height: 헤드라인 1.25, 본문 1.6
- font-weight 700+400 두 단계면 충분. 폰트 스택: Pretendard, 'Noto Sans KR', 'Malgun Gothic', sans-serif

## 색·배경 규칙
- 브랜드 팔레트에서: 배경 1색(또는 아주 미묘한 2색 그라디언트) + 텍스트 1색 + 액센트 1색. 3색 초과 금지
- 명도 대비: 텍스트/배경 최소 4.5:1. 큰 헤드라인은 3:1 허용
- 배경 장식: 기하 도형 2-3개(원/사선/반원)를 모서리에, 투명도 8-15%, 콘텐츠와 겹치지 않게
- 사진 흉내 금지 — 도형·그라디언트·패턴 일러스트만 (이 레인의 정체성)

## 안전 여백
- 모든 텍스트는 가장자리에서 96px 이상 (1080 기준)
- 9:16(스토리): 상단 240px·하단 320px은 UI 가림 구역 — 핵심 요소 배치 금지

## 출력 계약 (렌더러가 검증)
- 정확한 캔버스 크기: viewBox="0 0 W H" + width/height 명시
- 외부 참조 0 (이미지/폰트/스크립트) — self-contained SVG만
- 출력은 <svg…</svg> 코드만, 설명·코드펜스 금지
